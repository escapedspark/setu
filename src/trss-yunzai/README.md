# 云崽trss安装部署教程

### [trss官方文档](https://netlify.trss.me)

```
tips:warn
本教程仅适用于Linux系统及运行Linux系统的(云)服务器
请勿私自将本教程发至国内任何流量平台
```

1. 安装trss管理脚本

(云服务器玩家请先使用远程连接工具连接到您的服务器)

打开终端，输入命令并执行

```bash
bash <(curl -L gitee.com/TimeRainStarSky/TRSS_AllBot/raw/main/Install-Docker.sh)
```
